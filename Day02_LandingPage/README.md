# Dark 3D Landing Page - Defi Infrastructure

A premium, immersive landing page for a next-generation digital finance infrastructure project. Built with a futuristic dark-tech aesthetic, featuring glassmorphism, dynamic 3D-like animations, and neon glowing accents.

## 🚀 Key Features

-   **Cinematic Dark Theme**: Deep void background (`#080810`) with vibrant Neon Lime (`#B6FF1B`) and Electric Blue accents.
-   **Neon Data Stream**: Dynamic falling green streaks in the background for a deep-tech, high-energy environment.
-   **Glassmorphic UI**: High-end transparency and blur effects on all navigation panels and feature cards.
-   **Distinct Section Motifs**:
    -   **Performance Section**: Blue glow motif focusing on global scale and speed.
    -   **Security Section**: Neon green glow motif focusing on deep-tech protection.
-   **Micro-Animations**:
    -   Pulsing "breathing" neon borders on interactive elements.
    -   Parallax mouse-move effect on glow orbs.
    -   Smooth scroll hijacking for all local anchor links.
    -   Intersection Observer driven scroll-reveal animations.
-   **Modern Typography**: High-impact headlines using `Syne` and refined body text using `DM Sans`.
-   **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices.

## 🛠️ Tech Stack

-   **HTML5**: Semantic structure.
-   **CSS3**: Custom properties (tokens), Flexbox/Grid, and Keyframe animations.
-   **Vanilla JavaScript**: High-performance interactions and dynamic canvas-like effects.
-   **Google Fonts**: `Syne`, `DM Sans`, and `DM Mono`.

## 📁 Project Structure

-   `index.html`: The main structural file with all sections and functional IDs.
-   `style.css`: Comprehensive design system, animations, and glassmorphic styles.
-   `main.js`: Interactive logic, smooth scrolling, and background data stream generation.

## 💻 How to View

1.  Clone or download the repository.
2.  Open `index.html` in any modern web browser.
3.  Recommended: Use a local server (like VS Code Live Server) to see the smooth scrolling and animations as intended.
