A responsive personal portfolio website built with HTML, CSS, and JavaScript to showcase projects, skills, and provide contact information.

## 🚀 Live Demo

You can view the live demo here: **[https://your-username.github.io/portfolio](https://shreyashkingson.github.io/Day01_PortfolioPage/index.html)**


## ✨ Features

- **Responsive Design:** Adapts to any screen size, from mobile to desktop.
- **Project Showcase:** A dedicated section to display your work with links and descriptions.
- **Skills Section:** Highlight your technical skills.
- **About Me:** A section to introduce yourself.
- **Contact Info:** A way for visitors to get in touch.
- **Social Media Links:** Easy access to your professional profiles.

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**

## 📂 Project Structure

The project follows a simple and intuitive file structure:

```
portfolio/
│
├── index.html      # Main HTML file (the entry point)
├── style.css       # Main stylesheet for styling
├── script.js       # Main JavaScript file for interactivity
│
└── assets/
    ├── images/     # For project screenshots, profile picture, etc.
    └── icons/      # For favicons, social media icons, etc.
```
